# smalti docs

이 디렉터리는 smalti의 모든 인간 작성 문서(스펙·LLM 위키·브랜드·디자인)와 외부 원본을 모아 둡니다.

| 디렉터리 | 역할 |
|---|---|
| [`spec/`](./spec/) | PRD / TRD / UI 기능 명세 — 제품 결정의 단일 진실원 |
| [`brand/`](./brand/) | smalti 브랜드 아이덴티티 산출물 (전략·네이밍·버벌·비주얼·리뷰) |
| [`design/`](./design/) | UI 디자인 원본 (Pencil `.pen` 파일) |
| [`ideation/`](./ideation/) | 리브랜드·Rust 전환 등 큰 결정의 기획·선택 근거 |
| [`wiki/`](./wiki/) | LLM이 코드베이스를 탐색하며 누적해 온 위키 노트 |
| [`archive/`](./archive/) | 과거 핫픽스/버그픽스 메모 — 현행 지식과 분리 |
| [`raw/`](./raw/) | 외부 출처 원본 (읽기 전용) |
| [`assets/`](./assets/) | 문서에 첨부되는 이미지·다이어그램 |

번역: [한국어 README](./README.ko.md) — Phase 1에서 이동됨.
